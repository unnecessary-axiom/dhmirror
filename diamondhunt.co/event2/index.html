Diamond Hunt MMO


Following podcast 4, the mmo has been put back up online for a short period of time. Like event #1, the top 100 players will earn an event sigil. This event is completely optional and prizes related to the main game are only cosmetic. This event is really just for fun and allows players to see how the mmo was. The prizes are:

Rank 1 Player: Gold event sigil.
Rank 2 Player: Silver event sigil.
Rank 3 Player: Bronze event sigil.
Rank Top 100: Event Sigil.


Disclaimer:
The backend code of this game is old and unmaintained, so there is a possibility that the game crashes and becomes unplayable. If this were to happen, I will try to fix it without spending too much time. If the issue is too complex, the event will be canceled.
